# InSync Ecommerce CRM

## Environments

```
1. Production

==> Branch Name: main

2. Development

==> Branch Name: develop

```

## Install

```sh

git clone https://github.com/abendsoft/insync-crm.git
cd insync-crm
npm install
```

## Start development

```sh
npm run dev
```

## Build for production

```sh
npm run build
```

## Lint with

```sh
npm run lint
```
